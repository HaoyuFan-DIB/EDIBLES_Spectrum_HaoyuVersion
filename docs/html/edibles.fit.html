edibles.fit.fit.fit(star_name, data, model, silent=False, breakdown=False)[source]¶

A function that will fit a given multi-part model to a given spectrum.

Parameters:
  • star_name (string) – Name of the target star
  • data (tuple) – Spectrum data in the form (wave, flux)
  • model (object) – An unfit spectrum model
  • silent (bool) – If true, no plots will generate, defaults to False
Returns:

model that is fit to the data
Return type:

object
edibles.fit.fit.multifit(star_name, data_list, model_list, silent=False)[source]¶
A function that will fit 2 models to 2 spectra simultaneously.
This was created to fit the NaI doublets at ~3300 and ~5890 Angstroms.
Parameters:
  • star_name (string) – Name of the target star
  • data_list (tuple) – List of spectrum data in the form [(wave, flux), (wave, flux),…]
  • model_list (model instance) – A list of unfit spectrum models
  • silent (bool) – If true, no plots will generate, defaults to False
Returns:

models that are fit to the data
Return type:

list
